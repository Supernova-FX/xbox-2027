Legacy Mode

Legacy Mode preserves a refreshed version of the classic Xbox Series X|S OS for older games.

Full backwards compatibility: Runs Xbox One, Xbox 360, and original Xbox titles in their original environment.

Existing enhancements: Keeps features like FPS Boost, Auto HDR, and Quick Resume for supported titles.

Stable environment: Uses the same APIs and sandbox older games were built for.

Hybrid Mode

Hybrid Mode is a streamlined Windows‑based environment built for power and flexibility. It runs on a light Windows 11 foundation with the Xbox Full‑Screen Experience front and center. It supports third‑party stores like Epic, Steam, and EA, offers mod support and deeper settings, and lets developers target PC and Xbox with the same build.

Third‑Party Storefront Integration

Xbox Core X supports a flexible ecosystem of digital storefronts through Hybrid Mode, letting players access stores like Epic, Steam, and EA alongside the Microsoft Store. Purchases stay secure with system‑level entitlement management, and all games appear in one unified library. Legacy Mode ensures full compatibility with past Xbox titles, so that you are able to play your Xbox, Xbox 360, and Xbox One games.

Transitioning From Legacy To Hybrid

Over time, games move from Legacy Mode into Hybrid Mode. We test and certify older games to run in Hybrid Mode. Once ready, those games launch directly in Hybrid Mode instead of Legacy. Gradually, more of the library lives in the new environment.
