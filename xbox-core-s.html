Hybrid Mode

Hybrid Mode is a streamlined Windows‑based environment built for power and flexibility. It runs on a light Windows 11 foundation with the Xbox Full‑Screen Experience front and center. It supports third‑party stores like Epic, Steam, and EA, offers mod support and deeper settings, and lets developers target PC and Xbox with the same build.

Third‑Party Storefront Integration

Xbox Core S brings broad storefront support through Hybrid Mode, giving players access to Epic, Steam, EA, and more in addition to the Microsoft Store. All purchases use Xbox’s secure entitlement system and appear together in a unified library. Designed for the future, Core S focuses entirely on next‑gen Hybrid Mode.

Transitioning From Legacy To Hybrid

Over time, games move from Legacy Mode into Hybrid Mode. We test and certify older games to run in Hybrid Mode. Once ready, those games launch directly in Hybrid Mode instead of Legacy. Gradually, more of the library lives in the new environment.
